# Installation

Copy the theme folder to `/path/to/CTFd/themes/<theme_name>`. Then, in Admin Panel > Configuration > Theme, select the desired theme.

# Themes

1. [Watchdogs](watchdogs/README.md) (purple palette, Watchdogs font)
